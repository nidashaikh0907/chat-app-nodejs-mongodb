# 💬 Chat Application

A chat-style CRUD web application built using Node.js, Express.js, and MongoDB.  
This project demonstrates RESTful routing, server-side rendering, and basic message management functionality.

---

## 🚀 Features
- Create new messages  
- View all chat messages  
- Edit existing messages  
- Delete messages  
- Chat-style UI with message bubbles  
- Server-side rendering using EJS  

---

## 🛠 Tech Stack
- Node.js  
- Express.js  
- MongoDB  
- EJS (Embedded JavaScript Templates)  
- HTML5  
- CSS3

## ⚙️ Installation & Setup

Clone the repository  
git clone https://github.com/your-username/chat-app-nodejs-mongodb.git  

Navigate to the project folder  
cd chat-app-nodejs-mongodb  

Install dependencies  
npm install  

Start the server  
node index.js  

Open in browser  
http://localhost:8080/chats
